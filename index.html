Conversational Interface for Semiconductor Defect Detection using LLMs

AMLLab, Hansung University May 2024 – Jul. 2025

대형 언어 모델(LLM, Large Language Model)을 활용해 반도체 불량 검출을 위한 대화형 검사 인터페이스를 설계하고 구현하였습니다. 기존의 복잡한 GUI 기반 워크플로우를 자연어 명령 기반으로 전환하기 위해 자연어 명령 파이프라인(Natural Language Command Pipeline)을 구축하여, 사용자가 입력한 질의를 실시간으로 해석하고 검사 장비 제어 명령으로 매핑하는 시스템을 개발하였습니다.

프로젝트에서는 Transformer 기반 명령 분류 모델(Transformer-based Command Classification Model)을 설계하고 프롬프트 엔지니어링(Prompt Engineering)을 적용해 다양한 질의 유형에서도 높은 정확도를 확보하였습니다. 또한 RESTful API 서버를 직접 구현하여 검사 소프트웨어 및 장비와 연동하였고, 실제 인터페이스 제어 환경을 구성하였습니다.

실험 단계에서는 실제 반도체 패키징 공정에서 사용될 수 있는 데이터셋을 만들어 5-Fold 교차 검증(5-Fold Cross-Validation)과 Augmented Prompt 기법을 적용하였으며, 이를 통해 함수 호출 정확도와 모델의 일반화 성능을 검증하였습니다. 이 프로젝트는 자연어 기반 검사 자동화의 실효성을 입증하며, 지능형 작업자–설비 인터페이스(Intelligent Operator-Equipment Interface)의 가능성을 보여주었습니다.

한국디지털콘텐츠학회 2025년 하계종합학술대회 참가

발표 형식: 포스터 발표

반도체 패키지 불량 검출을 위한 Transformer 기반 명령 분류 모델을 활용한 피지컬 AI 시스템을 설계하고 구현했습니다. 본 시스템은 반도체 공정 환경에서 작업자가 입력한 자연어 명령을 실시간으로 해석하여 사전 정의된 제어 함수와 매개변수를 정확히 매핑하고 호출할 수 있도록 설계되었습니다. 이를 위해 사용자 입력 모듈, Transformer 기반 함수 추출기, 함수 반환 API로 구성된 계층형 아키텍처를 구축했습니다.


Exaone-2.4B, Exaone-7.8B, Trillion-7B 등 세 가지 Transformer 모델을 대상으로 5-Fold 교차 검증을 수행해 함수 호출 정확도와 처리 속도를 평가했습니다. 기본 프롬프트 환경에서의 성능을 측정한 후, 오답 사례를 포함한 Augmented Prompt 전략을 적용하여 성능을 비교 분석한 결과, 모델 전반에서 정확도와 안정성이 향상되었으며 특히 Exaone-2.4B는 함수 호출 정확도가 최대 15% 이상 개선되는 효과를 보였습니다.


본 연구는 자연어 기반 반도체 공정 제어의 가능성을 실증적으로 제시하며, 고정 UI 의존성을 탈피한 지능형 작업자–설비 인터페이스 설계 방향을 제안했습니다. 실험 결과는 반도체 제조 현장에서의 자동화 효율성 향상뿐만 아니라, 다양한 제조 환경에서 자연어 기반 피지컬 AI 시스템의 실용성을 입증하는 중요한 성과로 평가됩니다.

모바일 디바이스와 서버를 결합한 하이브리드 실시간 자세 분석 시스템을 직접 설계하고 제시했습니다. YOLOv11 포즈 추정 모델을 모바일 환경에 최적화하고, 볼링 동작을 기반으로 한 실시간 스코어 산출 알고리즘을 개발하여 자세 평가와 자연어 피드백을 구현했습니다.


기존 모바일 자세 피드백 시스템은 모바일 연산 한계와 서버 의존으로 자원 제약과 실시간성이 부족했습니다. 이를 해결하기 위해 온디바이스 추론과 서버 분석을 결합한 하이브리드 아키텍처를 설계·구현했습니다. YOLOv11 모델을 INT8(W8A8) 양자화해 모바일 환경에서 추론 속도와 메모리 사용량을 동시에 개선하고, 서버에서는 고해상도 분석을 통해 정확성을 확보했습니다.

어깨 각도 편차, 상체 이동 거리, 손목 누적 이동, 발목 변화 횟수 등 네 가지 지표를 기반으로 점수를 산출하고 이를 자연어 피드백으로 변환하는 알고리즘을 개발했습니다.

실험 및 결과

  • FP32 → INT8 최적화: 추론 시간 4.5ms → 1.1ms, 메모리 사용량 31MB → 26MB
  • 실시간성: 볼링 투구 시점 자동 판별 모바일 환경에서도 실시간 자세 분석 피드백 가능

성과

  • 모바일 환경에서 실시간 자세 분석 + 자연어 피드백 제공 가능성을 직접 입증
  • 스코어 산출 알고리즘을 통해 정량적 평가 + 직관적 피드백을 동시에 제공
  • 양자화 + 하이브리드 구조 설계로 다양한 실내 스포츠에 확장 가능한 모바일 시스템 구축

실시간 반도체 패키지 불량 검출을 위한 세부 관심 영역 자동 추출 기법

발행: 제32권 1호 (2025년)

저자: 임승택¹, 박영진², 최원용³, 한기준¹

소속: ¹한성대학교 컴퓨터공학부, ²디에스 선행기술연구소, ³제너셈 선행기술연구소

본 논문은 반도체 후공정 패키징 검사에서 필수적인 세부 ROI(Region of Interest)의 자동 추출 기법을 제안하며, 딥러닝 모델을 활용해 실시간 불량 검출 시스템의 정확성과 효율성을 동시에 향상시키는 것을 목표로 하였습니다.

연구 개요:
• 기존 rule-based ROI 설정의 수동 작업 및 오류 가능성 문제를 해결.
• Object Detection 모델을 적용하여 QFN/BGA 패키지의 Package, Lead, Pad, Ball 영역 자동 추출.
• 데이터셋 크기 변화와 이미지 augmentation 전처리 효과를 체계적으로 분석.
• 실제 양산 환경의 다양한 조명·노이즈 조건을 반영해 모델의 일반화 성능 검증.

주요 성과:
• YOLOv9 모델은 Precision 0.966, mAP50-95 0.925로 가장 안정적인 성능 기록.
• YOLOv11은 augmentation 적용 후 작은 Lead/Pad 객체 탐지에서 최대 +36% 성능 개선.
• 데이터 전처리와 증강 기법을 통해 제한된 데이터 환경에서도 실시간 ROI 추출 정확도 확보.

본 연구는 반도체 후공정 검사에서 기존 수동 티칭 기반 프로세스를 대체할 수 있는 딥러닝 기반 자동 ROI 추출 방식을 제시하고, 소량 데이터셋에서도 높은 탐지 성능을 확보할 수 있는 실험적 근거를 마련하였습니다. 이를 통해 실제 산업 현장에서 적용 가능한 실시간 불량 검출 시스템의 설계 방향을 제시합니다.

반도체 패키지의 불량 검출을 위한 딥러닝 기반 세부 패키징 구성 영역 추출 자동화

발표 형식: Poster Presentation

반도체 패키징 공정에서 결함 검출의 정확도를 높이기 위해, 딥러닝 기반 객체 탐지 모델을 활용하여 QFN 및 BGA 패키지의 세부 구성 요소(Package, Lead, Pad, Ball)에 대한 ROI(Region of Interest) 자동 추출 시스템을 연구하였습니다.

YOLO v8~v11 모델을 사용하여 객체 탐지 성능을 비교하고, 이미지 augmentation과 전처리 기법을 적용해 작은 소자(특히 Lead, Ball)의 검출 성능을 개선하였습니다.

실험 결과, YOLO v9 모델은 Precision 0.966, Recall 0.941, mAP50-95 0.925를 기록하며 전반적으로 우수한 성능을 보였고, YOLO v11 모델은 이미지 augmentation 적용 후 작은 소자 탐지에서 뚜렷한 mAP50-95 성능 향상을 보였습니다.

본 연구는 반도체 후공정 검사에서 자동화된 ROI 추출을 통해 작업 효율성을 높이고, 작은 구성 요소의 불량 검출 정확도를 개선하는 데 기여하였습니다.

• 딥러닝 기반 Object detection 모델을 활용하여 ROI 자동 추출 성능을 비교·분석.
• 실제 양산 라인의 QFN/BGA 패키지 데이터를 활용해 모델을 학습.
• 조명 환경과 노이즈 조건을 반영한 이미지 augmentation으로 작은 소자(Lead, Ball)의 검출 성능 강화.
• 이미지 cropping 및 전처리를 적용하여 모델 입력 최적화(640×640 기준).
• Precision, Recall, mAP50, mAP50-95 지표를 활용해 모델 성능을 정량적으로 평가.

ISMP-IRSP 2024 참가

Automatic ROI Recommender for Saw Singulation: Seamless Adaptation for Various Semiconductor Devices

발표 형식: Oral Presentation (공동 연구자, 2저자)

반도체 패키징 공정의 검사 항목이 다양해지고 복잡성이 증가함에 따라, 기존 rule-based 머신 비전 검사에서 발생하는 ROI(Region of Interest) 정의의 한계를 해결하기 위해 딥러닝 기반 ROI 추천 시스템을 연구하였습니다.

본 연구는 YOLO v7과 YOLO v11을 활용하여 Saw Singulation 공정의 다양한 패키지(Ball, Character, Barcode 등)에 대한 ROI 자동화 모델을 개발하고, 적은 데이터로도 빠른 적응이 가능한 few-shot 학습과 지속적 데이터 업데이트를 통한 Seamless Adaptation 방식을 제안하였습니다.

실험에서는 라운드별 학습 데이터 확장과 미학습 패키지 테스트를 수행하였으며, YOLO v7은 2D 코드에서 0.984 AP를 달성하고, 보이지 않는 데이터에서도 YOLO v7 0.529 AP, YOLO v11 0.452 AP를 기록하며 ROI 자동 추천 시스템의 가능성을 입증하였습니다.

이 연구는 반도체 후공정 검사에서 작업자의 수동 티칭 부담을 줄이고, 다양한 소자 변화에도 적응 가능한 지능형 비전 검사 시스템의 기반을 마련하는 데 기여하였습니다.
